Abstract

Relational structure, not only input–output maps

Φ

Existing symbolic regression approaches primarily learn explicit input–output mappings and often neglect relational structure among instances. Contrastive Symbolic Regression (CSR) integrates evolutionary feature construction with contrastive learning to shape a representation space where geometric proximity reflects similarity in the target space. A closed-form linear transformation aligns constructed features with targets; ridge regression captures global trends while adaptive KNN models local residuals; and DPP-based ensemble selection improves robustness via quality–diversity.

On 58 real-world regression datasets, CSR consistently surpasses traditional symbolic regression and modern machine learning counterparts.

Motivation

Why regression needs distance alignment

Classification can group discrete classes; regression needs a continuous geometry where nearby points share similar targets — the setting where KNN and instance-level explanations work.

Target-aligned representation
Φ(x): symbolic features  ·  y: target  ·  align pairwise distances, not only predictions
Method

Workflow of Contrastive Symbolic Regression

GP constructs symbolic features; closed-form contrastive alignment projects them into a target-aware space; ridge + KNN predict; DPP selects a diverse high-quality ensemble.

C1W

Closed-form W — contrastive projection without iterative gradients inside evolution.

C2K

Efficient LOOCV for K — one neighbor search selects neighborhood size for all K.

C3∑

Rank-weighted KNN — faithful fitness for representation smoothness.

C4◆

DPP ensemble — quality–diversity hall-of-fame selection.

III · Contrastive alignClosed-form W so feature distances track target distances.
IV–VI · Predict & ensembleRidge (global) + KNN (local) with DPP archive selection.
